# Asitha Wijerathne | Personal Portfolio

A modern, responsive personal portfolio website built to showcase my journey as a Software Engineering Undergraduate. It bridges the gap between my skills in full-stack web development and native Android application development.

![Portfolio Preview](public/profile.jpg) ## 🚀 Live Demo
[View Live Portfolio Here](https://your-portfolio-link.com) ## ✨ Key Features

- **Modern UI/UX:** Built with a custom "Midnight Blue" dark theme using Tailwind CSS.
- **Fully Responsive:** Mobile-first design that scales perfectly across tablets and desktop monitors.
- **Interactive Elements:** Features smooth CSS transitions, hover effects, and a dynamic fanned-out image gallery for project showcases.
- **Multi-Disciplinary Showcase:** Specifically structured to highlight projects across Web (React), Mobile (Java/Android), and Databases (SQL).

## 🛠️ Tech Stack (Portfolio)

This website was built entirely from scratch using:

- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel / Netlify / GitHub Pages] ## 💻 Featured Projects Included

1. **BudgetBuddy:** An offline-first Android personal finance tracker (Java, Room Database, Android SDK).
2. **StudyMate:** A collaborative web platform for student resource sharing (React, Node.js, Firebase).
3. **HospitalDB Manager:** A comprehensive relational database management system (MySQL, PHP).
